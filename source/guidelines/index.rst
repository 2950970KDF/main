LEITFÄDEN
=========
In diesem Abschnitt der Dokumentation finden Sie Anleitungen, die es ermöglichen ein System erfolgreich zu installieren.

Wollen Sie schnell ein funktionstüchtiges System Ihr eigen nennen,
dann würden wir Ihnen das virtualisierte System mit XEN
vorschlagen -> `Installationsleitfaden XEN <howto_installationsleitfaden/source/index.html>`__.


Sollten Sie stattdessen lieber genau wissen wollen, wie die Zusammenhänge bei der Installation verlaufen, dann legen wir Ihnen den Leitfaden zur Installation auf geeigneter Server-Hardware nahe.
Mit dieser Anleitung können Sie auch natürlich eine Installation auf eine andere Virtualierungslösung vornehmen, als die von uns favorisierte -> `Installationsleitfaden klassisch <howto_installation_klassisch/source/index.html>`__.

Mit den letzten beiden Leitfäden (`Linux <howto_linuxclients/source/index.html>`__ oder `Windows 10 <howto_windows10clients/source/index.html>`__) sind Sie in der Lage einen ersten Client in Ihr System zu integrieren. 

Danach können Sie Ihr neues System an Ihre Gegebenheiten anpassen. Anleitung dazu finden Sie in unseren Howtos.
